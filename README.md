# HL7 FHIR JP WG
HL7 FHIR Japan Working groupでの作業のために試行的に作成したリポジトリです。

このリポジトリの内容は個人的な研究・開発によるものであり、日本HL7協会
やHL7 FHIR実装検討ワーキンググループの見解を示すものではありません。
