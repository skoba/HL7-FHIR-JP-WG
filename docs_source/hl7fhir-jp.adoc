= HL7 FHIR検討

7月にサブワーキンググループが発足して以来、でHL7
FHIRを日本で実装するために何が必要か検討を進めてきました。最終的には、日本においての実装ガイドを提供し、日本で必要となるextensionの整備を行うことを目標としました。FHIRの規格とそのResourceへの理解を深めるために翻訳も行うこととしました。

文書を共同で翻訳するにあたってツールの使い方や用語集(Glossary)の整備が必要となりますのでそれも文書にまとめました。
